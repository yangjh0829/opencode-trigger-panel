name: keyword-trigger
description: 根据用户输入中的关键词自动匹配并加载对应 skills，支持多对多映射。持续预加载，每次对话生效。
---

你是一个关键词触发器。请执行以下步骤：

1. 读取同目录下的 `triggers.json` 文件，获取关键词 → skills 的映射规则
2. 检查用户当前输入是否包含任意规则中的关键词
3. 若匹配到多条规则，合并所有对应的 skills（去重）
4. 对每个匹配到的 skill，**使用 skill tool 加载该 skill**（不要手动 read 文件）。skill tool 是唯一的 skill 加载方式，调用后会自动注入 skill 指令
5. 如果匹配到多个 skills，依次调用 skill tool 加载每个 skill，将所有 skills 的指令合并执行，以最先匹配的 skill 为主流程，其余为辅

注意：
- 关键词匹配忽略标点符号和空格差异
- 一次用户输入可触发多个 skills
- 若无任何匹配，则按正常逻辑回复，不触发任何额外行为

## 匹配结果声明

当检测到关键词匹配时，**必须在回复开头**用以下格式声明匹配结果：

🎯 关键词匹配: [审核] [方案] → brainstorming, grill-me

然后再继续执行匹配到的 skill 的指令。

如果无匹配，不需要声明，直接正常回复。
